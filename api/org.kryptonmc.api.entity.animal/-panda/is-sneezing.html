abstract var isSneezing: Boolean

If this panda is currently sneezing.

Baby pandas have a 1/6000, or 0.000166666%, chance to sneeze on any tick. This is higher for weak baby pandas, at 1/500, or 0.002%. When a baby panda sneezes, all adult pandas within a 10 block radius will jump, and the baby panda has a 1/700 chance to drop a slimeball.
