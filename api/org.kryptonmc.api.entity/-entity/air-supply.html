airSupply

abstract val airSupply: Int

How much air this entity has, in ticks.

This will fill to a maximum of 300 in air, giving the entity a total of 15 seconds underwater before the entity begins to drown and die.

If this value reaches 0, and this entity is underwater, it will take 1 health point (half a heart) of damage for every second it remains underwater.
