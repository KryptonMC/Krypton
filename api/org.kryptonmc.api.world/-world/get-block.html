abstract override fun getBlock(x: Int, y: Int, z: Int): Block
Content copied to clipboard

Gets the block at the given coordinates.

This function will return the following in specific cases:

  • If the given y coordinate is greater than the maximum height of this world, this will return Blocks.VOID_AIR.

  • If there is no chunk loaded at the given coordinates (getChunkAt was null), this will return Blocks.AIR.

  • Else it will return the block at the given coordinates.
