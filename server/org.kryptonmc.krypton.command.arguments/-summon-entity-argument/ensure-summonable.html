fun ensureSummonable(key: Key): Key
Content copied to clipboard

Ensures the given key is both a valid entity type and the entity type that mapped to the key is summonable.

If the entity type is not summonable, ERROR_UNKNOWN_ENTITY will be thrown, despite the entity actually being known.
