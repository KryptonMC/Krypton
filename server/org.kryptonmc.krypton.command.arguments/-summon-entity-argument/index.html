SummonEntityArgument

object SummonEntityArgument : ArgumentType<Key>

An argument type that parses entity types that are summonable, in the form of namespaced keys.

The default namespace accepted here is "minecraft", just like with everywhere else that keys are used, and that is also valid for entity type keys here, so "pig" and "minecraft:pig" are equivalent.
