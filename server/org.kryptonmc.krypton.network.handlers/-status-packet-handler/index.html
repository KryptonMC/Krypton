StatusPacketHandler

class StatusPacketHandler(server: KryptonServer, val connection: NettyConnection) : PacketHandler

Handles all inbound packets in the Status state.

There are two packets in this state that we handle:

  • Status request - sent by the client to request status information

  • Ping - pings the server (to calculate latency on its end)
