LoginPacketHandler

class LoginPacketHandler(server: KryptonServer, val connection: NettyConnection, proxyForwardedData: ProxyForwardedData?) : PacketHandler

Handles all inbound packets in the Login state.

There are two inbound packets in this state that we handle:

  • Login Start - sent to initiate the login sequence

  • Encryption Response - sent to confirm the client wants to enable encryption
