NettyConnection

class NettyConnection(server: KryptonServer) : SimpleChannelInboundHandler<Packet> , NetworkConnection

A combination of the old channel handler and session classes. This handles both in one class, and offers minor improvements over the previous two, such as the absence of lateinit values.
