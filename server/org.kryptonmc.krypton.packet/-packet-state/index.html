enum PacketState : Enum<PacketState>

Represents one of the four packet states in the Minecraft protocol. Each one of these states has different packets and different purposes.

LOGIN

In the login state, the client attempts to login to the server, and this state establishes an encrypted connection with the client (when the server is in online mode). The server will also set a compression threshold for packet compression, if the threshold set in the config is 0.
