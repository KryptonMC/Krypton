SuggestionProviders

object SuggestionProviders

Holds built-in suggestion providers that are used client-side to make suggestions without bothering the server.

All retrievals in here will default to returning information about the special "ask_server" suggestion provider name, which instructs the client that they should bother the server for custom suggestions.
