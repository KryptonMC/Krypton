DefaultEntityTracker

class DefaultEntityTracker(entityViewDistance: Int) : EntityTracker

The default implementation of the entity tracker, used by the server.

Separating out the specification from the implementation lets us easily swap out the implementation if we want to, without having to change all the call sites.
